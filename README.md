# ShootingGame
![ShootingGame_sample](image/sample.gif)

## Overview

ジャンル：2Dドットシューティング

種別：フリーウェア

使用ライブラリ：[SDL](https://www.libsdl.org/)、[Game Programming in C++ Code](https://github.com/gameprogcpp/code)

## How to start

「ShootingGame.exe」を実行してください。

## How to play

- 矢印キー(←→)：移動

- Space：レーザー発射

- Esc：ゲーム終了
